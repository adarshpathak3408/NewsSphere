# News-Sphere - Personalized and Dynamic News Feed

NewsSphere is a sleek and responsive web application that provides the latest news from around the world in real-time. Powered by the GNews API, it allows users to search for specific topics, explore various categories, and enjoy a customizable news experience.

## Features

- **Real-Time News Updates**: Stay informed with up-to-date news from trusted sources.
- **Sorted Articles by Date**: News articles are arranged from latest to oldest, ensuring that the most recent updates are always displayed at the top.
- **Category-Based Navigation**: Filter news by categories such as Technology, Sports, India, Defence, and more.
- **Search Functionality**: Find specific news by entering keywords.
- **Dark Mode**: Toggle between light and dark modes for better readability.
- **Scroll-to-Top Button**: Easily navigate to the top of the page with a smooth scrolling effect.
- **Responsive Design**: Optimized for all screen sizes, ensuring a seamless experience on mobile, tablet, and desktop devices.


## üåê API Integration

This application integrates with the following API to fetch real-time news data:

### GNews API
- **Endpoint**: `https://gnews.io/api/v4/search?q={QUERY}&lang=en&country=us&max=10&apikey={API-KEY}`
- **Functionality**: 
  - **News Retrieval**: Fetches a list of news articles based on search queries.
  - **Article Details**: Provides detailed information such as title, description, publication date, and source.
  - **Filtering**: Supports filtering by language and country, allowing for localized news content.
  - **Pagination**: Limits the number of articles returned per request (e.g., up to 10 articles).

This API is utilized to ensure that users receive up-to-date and relevant news articles, enhancing the overall user experience.
